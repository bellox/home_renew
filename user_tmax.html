• TmaxSoft Operating System GUI Design (2008~2009)
    TmaxSoft 의 Operating System 의 Explorer Window GUI Design 으로, 다양한 방향으로 접근해본 프로젝트다. 기존 윈도와 크게 차별화 되는 부분은 없었으나 국내에서 한번도 시도해 보지 않았던 프로젝트의 GUI Design 을 경험할 수 있었다는 점으로도 충분한 메리트가 있었던 프로젝트라고 할 수 있다. 마이크로 소프트의 윈도7 수준의 그래픽을 구현하는 것을 목표로 그래픽 모티브를 설정하여 프로젝트를 시작 하였고, 외곽의 반투명 효과와, 부드러운 그림자효과 등을 그래픽이 아닌 프로그램으로 구현하도록 진행하였다.

  • TmaxSoft Mobile OS (2008~2009)
    TmaxSoft Mobile Operating System GUI Design 티맥스 고유의 모바일 OS 제작을 위한 컨셉 디자인 제안 작업을 진행하였고, 메인컨셉으로 Connecting Circle 을 도출하여 Look&Feel에 대한 요소로 적용 하였다. 메인 컬러톤은 그레이와 Peacock Green 톤을 기본으로 하여 사용자가 다양한 컬러를 선택하여포인트 컬러를 지정 할 수 있도록 하는것이 컬러 컨셉이었다. 다양한 방법으로 컬러를 선택하는 GUI 를 재미있게 했던 작업이다.
